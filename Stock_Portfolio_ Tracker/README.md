# 📈 Stock Portfolio Tracker

## 📌 Project Description

This project is developed for **CodeAlpha Python Programming Internship – Task 2**.

The program allows users to enter stock names and quantities. It calculates the total investment value using predefined stock prices and saves the result to a text file.

---

## 🚀 Features

* Enter stock name and quantity
* Calculates total investment
* Uses a predefined dictionary of stock prices
* Saves the result in `portfolio.txt`
* Beginner-friendly Python project

---

## 🛠 Technologies Used

* Python
* Dictionary
* Loops
* Conditional Statements
* File Handling
* Input/Output

---

## 📂 Project Structure

## ▶️ How to Run

1. Install Python.
2. Open the project folder.
3. Run the following command:

```bash
python stock_portfolio.py
```
