# 🤖 Basic Chatbot

## 📌 Project Description

This is a simple rule-based chatbot developed using Python for the CodeAlpha Python Programming Internship.

The chatbot responds to predefined user messages using basic `if-elif` conditions. It runs in the command line and continues the conversation until the user types **bye**.

---

## 🚀 Features

- Responds to "hello"
- Responds to "how are you"
- Tells its name
- Explains who created it
- Help command
- Exit using "bye"
- Handles unknown inputs

---

## 🛠️ Technologies Used

- Python
- Functions
- Loops
- If-Else Statements
- Console Input/Output

---

## ▶️ How to Run

1. Make sure Python is installed.
2. Save the file as `chatbot.py`.
3. Open Terminal or Command Prompt.
4. Run the following command:

```bash
python chatbot.py
```
