# 🎮 Hangman Game

## 📌 Project Description

This project is developed for **CodeAlpha Python Programming Internship – Task 1**.

It is a simple text-based Hangman Game where the player guesses a hidden word one letter at a time. The player has only **6 incorrect guesses** to find the correct word.

---

## 🚀 Features

* Random word selection
* 5 predefined words
* 6 chances to guess
* Displays guessed letters
* Win and Lose conditions
* Beginner-friendly code

---

## 🛠 Technologies Used

* Python
* Random Module
* While Loop
* If-Else Statements
* Lists
* Strings
